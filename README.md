## Calliope mini V3 Demo Hex

Diese Datei enthält die vier Startprogramme für den Calliope mini V3.
Im Gegensatz zum originalen Programm kann dieses in MakeCode geöffnet und verändert werden.

Dieses Programm ist besonders als Startprogramm für iOS-Geräte geeignet, weil die Bluetoothfunktionalität 
besonders für die folgende Kommunikation über die iOS-App eingestellt ist.
Partial Flashing ist integriert, somit kann ein folgendes Programm aus MakeCode besonders schnell übertragen werden.

In diesem Programm befindet sich nicht die Testroutine aus der Produktion.
